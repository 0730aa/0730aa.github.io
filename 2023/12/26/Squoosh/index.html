搭建在线图片无损压缩 Squoosh

发表于2023-12-26|更新于2023-12-30|教程分享
|字数总计:154|阅读时长:1分钟|阅读量:

docker 安装

  • 直接 docker run
  • run 完之后访问 ip:8000

    添加一张图片，如我这张图大小为 3.25MB

    进行压缩后，拉动右边 quality 可以快捷调整图片大小，如我调成 15，从图片左右两边对比没有太大差异的，但是他的文件大小已经从 3.25mb 降到 223kb，压缩率达到 93%
  • 当然右边还有更高级的编辑，点击 resize 可以调分辨率之类的.
    调节完之后直接点击右边那个下载按钮就得到压缩后的图片
